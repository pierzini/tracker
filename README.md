# tracker

## Help

```
tracker

USAGE:
    tracker [OPTIONS]

FLAGS:
        --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -b, --browser <browser>
    -f, --file <cfgfile>
    -h, --host <host>           [default: localhost]
    -i, --index <index>
    -p, --port <port>           [default: 9200]
```

## Installation

On macOS or GNU/linux, run:

```
$ ./install.sh
```
